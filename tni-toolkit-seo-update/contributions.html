How This File Works

This is a simple changelog-style record of who contributed what and when. It complements:

  • credits.html ‚Äî Styled credits page on the toolkit site
  • JSON _meta.contributors ‚Äî Attribution embedded in data files
  • HTML header comments ‚Äî Attribution embedded in tool files

Adding entries

  • Group by date, then by contributor
  • Version-bumped files: `filename` vX.Y.Z ‚Äî Brief description
  • General work: one-liner describing the change
  • Keep it concise ‚Äî details live in commit messages and file headers
